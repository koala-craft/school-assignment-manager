## 新規タスクの作成

**新規タスクを作成する際は [template.md](./template.md) を参考にしてください。** テンプレートに含まれる以下を毎回含めること。

- 概要（設計書参照、エンドポイント、権限）
- 状態・優先度
- なぜ未実装か
- 実装工数の目安
- 実装するとできること
- 実装状況（完了時）
- 追加対応（必要な場合）
- 権限・依存関係・技術メモ（必要な場合）
