# Step 12: データインポート

## 概要

CSVによる一括インポート機能を実装する。APIと画面の両方が必要。

## 実装内容

### API（バックエンド）

| 項目 | 内容 |
|------|------|
| 学生一括インポート | POST /admin/import/students |
| 履修一括インポート | POST /admin/import/enrollments |
| 提出物一括インポート | POST /admin/import/assignments（任意） |
| バリデーション | 必須項目、形式、重複チェック |
| エラーレポート | エラー行の特定、詳細表示 |

### 画面（フロントエンド）

### A-010: データインポート画面

| 項目 | 内容 |
|------|------|

| 機能 | インポート種類選択、ファイルアップロード、エラーレポート表示 |
| 種類 | 学生、履修 |
