## 実装内容

### SA-001: 管理者学生ダッシュボード

| 項目 | 内容 |
|------|------|
| URL | `/student-admin/dashboard` |
| API | GET /dashboard/teacher または専用エンドポイント（担当科目フィルタ） |
| 機能 | 担当科目サマリ、未提出者数、クイックアクセス、最近の更新履歴 |

### SA-002: 提出状況更新画面

| 項目 | 内容 |
|------|------|
| URL | `/student-admin/submissions` |
| API | GET /admin/submissions、更新API（権限制限） |
| 機能 | 提出状況マトリクス（T-009と類似）、未提出→提出済み、再提出待ち→再提出済み のみ更新可能 |
| 制限 | 採点・コメント・再提出指示は不可 |

## 権限の違い

| 操作 | 教員 | 管理者学生 |
|------|------|-----------|
| 未提出→提出済み | ✓ | ✓ |
| 再提出待ち→再提出済み | ✓ | ✓ |
| 採点 | ✓ | ✗ |
| 再提出指示 | ✓ | ✗ |
